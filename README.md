# Superstore_Performance_Dashboard
1. Project Title / Headline📊 Superstore Performance Analysis Dashboard (Power BI Project

2. Short Description / PurposeThis project provides a deep-dive diagnostic analysis of retail "Superstore" operations to uncover key drivers of profitability and sales growth. The dashboard transforms raw transactional data into actionable insights, helping stakeholders identify underperforming regions, optimize product categories, and understand customer purchasing patterns across the United States.

3. Tech StackPower BI: For report authoring and advanced visual construction.Power Query: Used for data cleaning and ETL processes to handle complex datasets.DAX (Data Analysis Expressions): To create custom measures including YTD growth and profit margins.Data Modeling: Implementation of star schema relationships for optimized report performance.

4. Data SourceDataset: US Superstore Sales Dataset.Format: Power BI Template File (.pbit) with underlying metadata structures.Contains: Order details, Product Category/Sub-category, Regional Geography (5 Regions), Sales, Quantity, Discounts, and Profit metrics.

5. Features & Highlights📈 Executive Sales Overview: Real-time tracking of total sales and profit margins across multiple segments.🗺️ Geographic Drill-Down: Interactive maps visualizing performance trends across 5 key regions.📦 Category Performance: Identification of high-return vs. loss-making product sub-categories.📅 Time-Series Analysis: Monthly and daily sales trends to detect seasonal fluctuations.💡 Discount Impact Study: Analyzing how heavy discounting affects the overall net profit
